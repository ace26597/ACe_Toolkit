# Real Estate Due Diligence Checklist

## Property: ____________________
## Address: ____________________
## Inspection Period: ____ days (ends: ________)

---

## 1. FINANCIAL REVIEW

### Income Verification
- [ ] Obtain rent roll from seller
- [ ] Verify current leases (copies of all leases)
- [ ] Review tenant payment history (12 months)
- [ ] Confirm security deposit amounts
- [ ] Check for any rent concessions or side agreements
- [ ] Verify other income sources (laundry, parking, etc.)

### Expense Verification
- [ ] Property tax bills (2-3 years)
- [ ] Insurance declarations page
- [ ] Utility bills (12 months) - water, electric, gas, trash
- [ ] HOA statements if applicable
- [ ] Property management agreement
- [ ] Service contracts (landscaping, pest, etc.)
- [ ] Repair and maintenance records

### Financial Analysis
- [ ] Calculate actual NOI from verified numbers
- [ ] Compare to seller's pro forma
- [ ] Stress test with vacancy increase
- [ ] Model rent increase potential
- [ ] Calculate cap rate, GRM, cash-on-cash
- [ ] Run 5-year projection

---

## 2. PHYSICAL INSPECTION

### Professional Inspections
- [ ] General home inspection (schedule by: ______)
- [ ] Roof inspection if >10 years old
- [ ] HVAC inspection
- [ ] Plumbing camera scope (older buildings)
- [ ] Electrical inspection (if knob-and-tube or aluminum)
- [ ] Foundation inspection (if any concerns)
- [ ] Pool/spa inspection if applicable
- [ ] Pest/termite inspection

### Property Walk-Through
- [ ] Exterior condition (siding, paint, windows)
- [ ] Roof condition (visible damage, age)
- [ ] Foundation (cracks, settling)
- [ ] Drainage and grading
- [ ] Parking areas and driveways
- [ ] Landscaping condition
- [ ] Common areas
- [ ] All unit interiors
- [ ] Basement/crawlspace
- [ ] Attic access and condition

### Systems Checklist
- [ ] HVAC age and condition
- [ ] Water heater age and condition
- [ ] Electrical panel capacity and condition
- [ ] Plumbing type (copper, PEX, galvanized)
- [ ] Appliances included and condition
- [ ] Smoke/CO detectors
- [ ] Fire extinguishers

---

## 3. LEGAL AND TITLE

### Title Review
- [ ] Order preliminary title report
- [ ] Review for liens and encumbrances
- [ ] Verify legal description
- [ ] Check easements and CC&Rs
- [ ] Review survey (or order new one)
- [ ] Confirm property boundaries
- [ ] Check for any pending litigation

### Zoning and Permits
- [ ] Verify current zoning
- [ ] Check permitted use
- [ ] Review building permits history
- [ ] Identify any unpermitted work
- [ ] Check for code violations
- [ ] Certificate of occupancy

### Lease Review
- [ ] Review all lease agreements
- [ ] Note lease expiration dates
- [ ] Identify tenant rights (first refusal, etc.)
- [ ] Check for assignability
- [ ] Review any amendments
- [ ] Verify security deposit handling

---

## 4. ENVIRONMENTAL

- [ ] Phase I Environmental Assessment (if commercial/industrial)
- [ ] Lead paint disclosure (pre-1978)
- [ ] Asbestos survey if applicable
- [ ] Radon test
- [ ] Mold inspection if concerns
- [ ] Underground storage tank search
- [ ] Flood zone determination

---

## 5. MARKET ANALYSIS

- [ ] Comparable sales analysis (last 6 months)
- [ ] Comparable rentals analysis
- [ ] Neighborhood walk-through
- [ ] Crime statistics review
- [ ] School ratings if applicable
- [ ] Employment and economic trends
- [ ] Future development plans
- [ ] Talk to neighbors if possible

---

## 6. INSURANCE

- [ ] Get insurance quotes
- [ ] Check for claims history on property
- [ ] Verify flood zone status
- [ ] Determine if special coverage needed
- [ ] Confirm coverage requirements for loan

---

## 7. FINANCING

- [ ] Loan pre-approval in place
- [ ] Appraisal ordered (by: ______)
- [ ] Lender due diligence requirements
- [ ] Rate lock deadline
- [ ] Final loan commitment deadline
